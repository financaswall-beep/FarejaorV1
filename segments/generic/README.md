# Segmento Generic

Segmento neutro e genérico do Farejador.

Contém regras de linguagem que fazem sentido em pelo menos 3 verticais:
- reclamação de preço;
- sinal de urgência;
- menção a concorrente;
- sinal positivo/fechamento;
- sinal de abandono.

Não contém regras específicas de nenhum segmento de negócio.

## Arquivos

- `rules.json` — regras declarativas executáveis;
- `lexicon.json` — sinônimos e stopwords;
- `scenarios.json` — exemplos de validação (não usado pelo motor em runtime).
