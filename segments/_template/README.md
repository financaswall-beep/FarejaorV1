# Segmento Template

Template mínimo para criar novos segmentos.

Não contém regras de negócio reais. Serve como estrutura de prova e referência.

## Como usar

1. Copiar esta pasta para `segments/<nome>/`;
2. Preencher `rules.json` com regras do segmento;
3. Atualizar `lexicon.json` com vocabulário específico;
4. Adicionar cenários em `scenarios.json` para validação;
5. Registrar rota em `segments/routing.json`.

## Locale

pt-BR (ajustar conforme necessidade do segmento).
